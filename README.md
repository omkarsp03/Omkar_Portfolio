# Omkar Patil | Software Developer Portfolio

Welcome to the source code for my personal portfolio website! This project is a showcase of my skills, experiences, and projects as a Computer Science undergraduate, Full-Stack Developer, and AI/ML enthusiast. It features a modern, interactive design built with vanilla web technologies, Three.js, and GSAP.

## ✨ Features

- **Dynamic 3D Background**: Implemented using **Three.js** to create an interactive particle and geometric mesh environment that responds to mouse movement and scrolling.
- **Smooth Animations**: Powered by **GSAP** for performant scroll-triggered animations, typing effects, and hover interactions.
- **Custom UI Elements**: Custom cursor (following dot and ring), custom scroll progress indicator, and a sleek preloader.
- **Light & Dark Mode**: Seamless theme switching that dynamically updates CSS tokens and Three.js background colors.
- **Responsive Design**: Fully responsive layout optimized for all devices, complete with a mobile-friendly hamburger menu navigation.
- **Immersive Sections**: 
  - **Hero**: A compelling introduction with a typing effect and key statistics.
  - **About & Experience**: Highlighting my professional and educational journey.
  - **Skills**: Detailed technical arsenal broken down into categories.
  - **Projects & Hackathons**: Highlighting featured AI, Healthcare, Full-Stack, and Web Development creations.

## 🛠️ Technologies Used

- **HTML5**: Semantic and accessible markup.
- **CSS3**: Vanilla CSS with modern design tokens, CSS variables, flexbox/grid, and complex gradients. No CSS frameworks are used to maintain full custom design control.
- **JavaScript (ES6+)**: Core logic for interactions, animations, and DOM manipulation.
- **Three.js**: Used for rendering the interactive 3D WebGL background and particle system.
- **GSAP (GreenSock Animation Platform)**: Used for high-performance animations, scroll effects, and hover transitions.

## 🚀 Getting Started

This is a frontend website using ES6 modules. To run it locally and without CORS issues, you should serve it via a local web server.

### Running Locally

1. **Clone the repository:**
   ```bash
   git clone https://github.com/omkarsp03/Omkar_Portfolio.git
   cd Omkar_Portfolio
   ```

2. **Serve the project:**
   Using Node.js:
   ```bash
   npx serve .
   ```
   Or using Python:
   ```bash
   python3 -m http.server 3000
   ```

3. **Open your browser:**
   Navigate to `http://localhost:3000` (or the port specified by your server) to view the portfolio.
